- 如何评价算法的好坏？
  - 时间复杂度  执行时间
  - 空间复杂度 占用内存空间

- 时间复杂度怎么计算
  - 计算执行步骤 T（n） T(n) 表示算法在输入规模为 n 时的运行时间（或基本操作的执行次数），是用来描述算法时间复杂度的函数。并进一步简化为大O表示法（如 O(n)、O(n²) 等）来刻画其增长趋势。
  - 算法的时间复杂度，不是展现逻辑代码，而是随着输入规模的增大，反应趋势。
    抓主要矛盾

## 时间复杂度
O（1） 常数时间复杂度 O(log n) 对数时间复杂度 O (n) 线性时间复杂度 
O(n log n) 线性对数时间复杂度 O(n²) 平方时间复杂度 O(2^n) 指数时间复杂度
